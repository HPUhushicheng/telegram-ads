# 克隆仓库
```
git clone https://github.com/HPUhushicheng/telegram-ads.git
```
# 进入项目目录
```
cd telegram-ads/code-2
```
# 安装项目依赖
```
pnpm install 
```
# 运行项目
```
pnpm run dev
```
